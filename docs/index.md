## Introdução

O projeto tem como propósito o desenvolvimento de uma aplicação completa baseada em computação em nuvem, integrando diferentes serviços da Amazon Web Services (AWS) para construir uma solução escalável e segura. 

A aplicação consiste em uma Área de Clientes capaz de realizar o upload, armazenamento, processamento e exibição de imagens, utilizando componentes como Amazon S3, EC2, Lambda, DynamoDB e API Gateway. O projeto visa não apenas a implementação técnica, mas também o domínio de boas práticas de arquitetura em nuvem, automação e documentação detalhada de toda a infraestrutura.
